%% myTables.tex
% A common file to store all table definitions 
%
% In preparing your thesis, one of the first things you should do is
% organize your data.  Then, one of the last things you'll do is
% reorder your figures and tables so they display where you want them
% to in the text.  Organizing tables definitions in a common files
% helps:
%
%   1. Write new tables using earlier examples.
%
%   2.  Isolate code and minimize the risk of introducing bugs in the
%   final editing process.  Trust me, moving around just one line of
%   code is easier.
%
%   3.  Reuse tables in other papers.  <=== the best reason!
%
% Note command names can not include numbers and special characters.
%
% To make the file more searchable, use naming conventions that map 
% data set to the command name \tablabData to the table's label tab:labData.
