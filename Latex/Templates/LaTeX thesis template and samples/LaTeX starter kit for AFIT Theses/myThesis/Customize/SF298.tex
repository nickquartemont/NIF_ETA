draft of their thesis using \LaTeX\xspace.  The primer is produced
according the tenets described within the document.  All source code
is provided in a zip file posted to

structure of this zip file demonstrates a practical way to organize a
thesis with its supporting materials and---further---illustrates how
your document can be produced with version control.  }
